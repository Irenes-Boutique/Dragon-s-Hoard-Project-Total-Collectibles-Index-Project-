# Total Collectibles Index Project

This GitHub repo contains the code for the paper "" written by Irene Chung and Professor Jack Davis at the University of Waterloo. This repo contains all the relevant code for data collection, cleaning, and analysis. 

There are 3 folders labelled Data Collection, Data Cleaning, and Analysis that contain the code for each part of the paper. Each Jupyter Notebook is labelled relative to what the code in the notebook does (example: AMPI Index.ipynb in the Data Cleaning folder is the notebook where the data for the AMPI index is tranformed and sorted).
